# 赛道二参考项目

与「忆青集」类似的视觉理解 / 多模态 / AI 叙事开源项目，供对照学习。

## 一键 Fork 到你的 GitHub

```powershell
# 1. 登录（只需一次）
gh auth login

# 2. 执行 fork 脚本
.\scripts\fork-reference-repos.ps1
```

Fork 后会出现在：https://github.com/cccyyylll888

## 项目清单

| 优先级 | 原仓库 | 学什么 |
|--------|--------|--------|
| ⭐⭐⭐ | [AI-StoryWeaver](https://github.com/SannidhyaDas/AI-StoryWeaver) | 多图上传 → Gemini 写故事 → TTS 朗读 |
